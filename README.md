# 《28天攻克PMP》勘误

## 规范

**注意**:

1. 本勘误表，以第58期PMP培训班所发讲义资料的章节及页数为准。
1. 为便于管理，《28天攻克PMP》的勘误每个章节一个文件，每套模拟题一个文件，问题应按出现位置的顺序填写在相应的文件中。
1. 每个问题的格式应参照如下例子：

```text
## 1.4 项目组合管理、项目集管理和项目管理的关系及组织项目管理 // 错误所在的章节编号和名称

- 位置: 第 3 页本小节第 1 段  // 错误所在的位置
- 错误内容: 项目及的目标      // 出现错误的内容
- 错误原因: 错别字           // 出错的原因
- 应更正为: 项目集的目标      // 错误所应更正为的内容
```
